# إصلاح عرض الأخبار العربية - ملخص تنفيذي

## 🎯 المشكلة المحددة

الترجمة العربية للأخبار **أصبحت غير واضحة ومكسورة وغير صحفية**:

```
❌ ظهور HTML entities مثل: &#039; &amp; &quot;
❌ عناوين حرفية سيئة جداً وغير مفهومة
❌ نصوص مشوّهة ممزوجة برموز غريبة
❌ لا توجد قاعدة جودة قبل العرض
❌ اختلاط الأحرف اللاتينية مع العربية
```

## ✅ الحل المطبق

### 1. Pipeline معالجة شامل للأخبار

**خطوات المعالجة:**
```
النص الخام
↓
[1] فك HTML entities (&#039; → ')
↓
[2] تنظيف النص (إزالة HTML tags, URLs, رموز خاصة)
↓
[3] فحص الجودة (هل النص آمن للعرض؟)
↓
[4] عرض مباشر (إذا جودة عالية)
    أو
    توليد عنوان صحفي جديد (إذا جودة منخفضة)
↓
نص عربي واضح وصحفي
```

### 2. توليد عناوين صحفية حقيقية (لا ترجمة حرفية)

بدل محاولة ترجمة النص الرديء، نستخرج المعنى ونُعيد الصياغة:

```
مثال 1:
بدل:  "ترامپ عند مفترق طرق وأمريكا تتعامل مع..."
استخدم: "ترامب يدرس خيارات تصعيدية جديدة تجاه إيران"

مثال 2:
بدل:  "أسعار الزيت تزيد 5% مع حالة عدم اليقين..."
استخدم: "أسعار النفط تشهد تطورات جديدة وسط التوترات العالمية"

مثال 3:
بدل:  "روسيا أطلقت مواد حربية قاتلة على هدف..."
استخدم: "روسيا تشن هجومًا بالمسيّرات على أوكرانيا"
```

### 3. فحوصات جودة قبل العرض

قبل عرض أي عنوان/ملخص:

✅ **HTML entities:** يتم فكها تلقائيًا
✅ **الأحرف الخاصة:** إزالة الرموز المشوّهة
✅ **اللغة المختلطة:** تجنب العربية الممزوجة بالإنجليزية  
✅ **الطول:** اختصار النصوص الطويلة
✅ **المعنى:** التأكد من الوضوح والمفهومية

## 🧪 كيفية الاختبار

### اختبار سريع:

1. اذهب إلى [صفحة الأخبار](/news)
2. غيّر اللغة إلى العربية
3. تحقق من:
   - ✅ عدم ظهور `&#039;` أو `&amp;`
   - ✅ عناوين واضحة وليست حرفية
   - ✅ ملخصات موجزة ومفهومة

### اختبار برمجي:

```javascript
// في console
import { processNewsItem } from './lib/i18n/summaryLocalizer';

const messyNews = {
  title: "Trump&#039;s latest decision amid US-Iran tensions",
  summary: "The president...",
  source: "Reuters"
};

const clean = processNewsItem(messyNews, "ar");
console.log(clean.title); // عنوان صحفي عربي نظيف
console.log(clean.qualityScore); // "high" أو "medium"
```

## 🔄 سير العملية

```
عند عرض خبر عربي:

1. هل النص يحتوي على HTML entities أو رموز خاصة؟
   ├─ نعم → استدعِ processNewsItem()
   └─ لا → استخدم localizeSummaryText (أسرع)

2. في processNewsItem():
   ├─ فك entities
   ├─ تنظيف
   ├─ فحص الجودة
   ├─ إذا جودة عالية → اعرض النص
   └─ إذا جودة منخفضة → وَلِّد عنواناً صحفياً

3. أعِد النص الجاهز للعرض
```
